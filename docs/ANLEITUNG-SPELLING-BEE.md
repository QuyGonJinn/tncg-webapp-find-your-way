# Anleitung für Spelling Bee – Gebärden-Rätsel

## 🐝 Willkommen zu Spelling Bee!

**Spelling Bee** ist eine spezielle Station der Glaubensjagd, bei der ihr **Gebärden-Videos** anschauen und Wörter erraten müsst. Es ist eine passive Station, die ihr jederzeit während des Spiels erledigen könnt.

**Punkte:** +20 XP  
**Dauer:** ca. 10-15 Minuten  
**Besonderheit:** Separate Webseite mit Videos  
**Schwierigkeit:** Mittel (mit Gebärden-Alphabet)

---

## 1️⃣ Vorbereitung – Was ihr braucht

### Geräte & Materialien

- **Smartphone, Tablet oder Computer** mit Internetverbindung
- **Gebärden-Alphabet** (ausgedruckt oder digital)
- **Zettel und Stift** (optional, zum Notieren)
- **Ruhe und Konzentration** 😊

### Gebärden-Alphabet

Das Gebärden-Alphabet wird euch bei der Station ausgehändigt. Es zeigt, wie jeder Buchstabe in Gebärdensprache aussieht.

**Beispiel:**
```
A = Faust mit Daumen nach oben
B = Flache Hand mit Fingern zusammen
C = Hand wie ein C geformt
...
```

---

## 2️⃣ Spelling Bee öffnen

### Schritt 1: Zur Station gehen

1. **Findet die Spelling Bee Station im Gelände**
   - Fragt einen Admin, wo die Station ist
   - Dort liegt ein **Zettel mit einem QR-Code** oder einer **URL**

2. **QR-Code scannen oder URL eingeben**
   - Öffnet die Kamera-App auf eurem Handy
   - Scannt den QR-Code
   - Oder: Gebt die URL in den Browser ein
   - Beispiel: `https://fyw.tncg.de/spelling-bee`

### Schritt 2: Mit eurem Team-Code anmelden

1. **Die Spelling Bee Seite öffnet sich**
   - Ihr seht einen Login-Bildschirm
   - Es wird euer **Team-Code** verlangt

2. **Euren 4-stelligen Team-Code eingeben**
   - Gebt euren Team-Code ein (z.B. `A3KX`)
   - Das ist der Code, den ihr beim Erstellen des Teams bekommen habt
   - **Wichtig:** Großbuchstaben verwenden!

3. **"🚀 Anmelden" drücken**
   - Klickt auf den Button
   - Die Seite lädt kurz
   - Ihr seht dann die 5 Videos

---

## 3️⃣ Die 5 Videos anschauen

### Überblick

Ihr seht **5 Videos** nebeneinander (oder untereinander auf dem Handy). Jedes Video zeigt ein Wort in Gebärdensprache.

```
Video 1: GLAUBE
Video 2: HOFFNUNG
Video 3: LIEBE
Video 4: FREUDE
Video 5: FRIEDE
```

### Schritt-für-Schritt: Ein Video analysieren

1. **Video anschauen**
   - Klickt auf den Play-Button
   - Schaut euch die Gebärden genau an
   - Achtet auf die Handbewegungen und Positionen

2. **Video mehrmals anschauen**
   - Klickt auf Play, um das Video nochmal zu sehen
   - Schaut es 2-3 Mal an
   - Jedes Mal könnt ihr neue Details bemerken

3. **Gebärden mit dem Alphabet vergleichen**
   - Nehmt das Gebärden-Alphabet zur Hand
   - Vergleicht die Gebärden im Video mit dem Alphabet
   - Schreibt die Buchstaben auf (z.B. `G-L-A-U-B-E`)

4. **Das Wort erraten**
   - Setzt die Buchstaben zusammen
   - Welches Wort ergibt sich?
   - Beispiel: `G-L-A-U-B-E` = `GLAUBE`

### 💡 Tipps zum Verstehen

| Tipp | Erklärung |
|------|-----------|
| **Langsam anschauen** | Schaut das Video mehrmals an, um alle Buchstaben zu sehen |
| **Alphabet nutzen** | Vergleicht jeden Buchstaben mit dem Alphabet |
| **Zusammen überlegen** | Diskutiert mit eurem Team, welche Buchstaben ihr seht |
| **Logisch denken** | Welche Wörter ergeben Sinn? (z.B. religiöse Begriffe) |
| **Nicht aufgeben** | Manche Gebärden sind schwierig – versucht es nochmal! |

---

## 4️⃣ Die 5 Wörter eingeben

### Eingabefelder

Unter jedem Video seht ihr ein **Eingabefeld** mit der Beschriftung "Wort eingeben...".

### Schritt-für-Schritt: Ein Wort eingeben

1. **Auf das Eingabefeld klicken**
   - Klickt unter dem Video in das Eingabefeld
   - Der Cursor blinkt

2. **Das Wort eingeben**
   - Gebt das Wort ein, das ihr erraten habt
   - Beispiel: `GLAUBE`
   - **Wichtig:** Großbuchstaben verwenden!

3. **Feedback erhalten**
   - Wenn das Wort richtig ist: ✅ **Grüner Haken**
   - Wenn das Wort falsch ist: ❌ **Roter Haken**
   - Ihr seht auch: "✓ 1 / 5 Wörter richtig"

4. **Falsches Wort? Nochmal versuchen!**
   - Löscht das Wort
   - Schaut das Video nochmal an
   - Versucht es nochmal

### Alle 5 Wörter eingeben

Wiederholt die Schritte für alle 5 Videos:

1. Video 1 anschauen → Wort eingeben
2. Video 2 anschauen → Wort eingeben
3. Video 3 anschauen → Wort eingeben
4. Video 4 anschauen → Wort eingeben
5. Video 5 anschauen → Wort eingeben

---

## 5️⃣ Alle Wörter richtig? 🎉

### Wenn alle 5 Wörter korrekt sind

1. **Bestätigungsmeldung erscheint**
   - Ihr seht: "🎉 Alle Wörter richtig!"
   - Eine grüne Box mit Glückwunsch-Animation

2. **Der Code wird angezeigt**
   - Ihr seht einen **4-stelligen Code** (z.B. `SB95`)
   - Die Meldung sagt: "Hier ist dein Code:"
   - Darunter: "Trage diesen Code in die App ein!"

3. **Code merken oder aufschreiben**
   - Schreibt den Code auf einen Zettel
   - Oder macht ein Foto vom Bildschirm
   - Ihr braucht diesen Code gleich!

---

## 6️⃣ Code in der Hauptapp eingeben

### Zurück zur Hauptapp

1. **Geht zurück zur Hauptapp**
   - Öffnet einen neuen Browser-Tab
   - Oder klickt auf den "Zurück"-Button
   - Ihr seht wieder die Stationen-Übersicht

2. **Findet die "Spelling Bee" Station**
   - Scrollt durch die Stationen
   - Sucht nach der Station mit dem 🐝 Emoji
   - Klickt auf die Karte

### Code eingeben

1. **"Code eingeben" drücken**
   - Klickt auf den Button "Code eingeben"
   - Ein Eingabefeld erscheint

2. **Den Code eingeben**
   - Gebt den Code ein, den ihr von Spelling Bee bekommen habt
   - Beispiel: `SB95`
   - **Wichtig:** Großbuchstaben verwenden!

3. **"Bestätigen" drücken**
   - Klickt auf "Bestätigen"
   - Die App überprüft den Code

4. **✅ Station freigeschaltet!**
   - Die Karte wird grün
   - Ihr seht: "+20 XP" mit Animation
   - Der Fortschrittsbalken aktualisiert sich
   - **Glückwunsch! Ihr habt Spelling Bee erledigt!** 🎉

## 7️⃣ Häufige Fragen & Probleme

### "Ich verstehe die Gebärden nicht"

**Lösung:**
- Schaut das Video mehrmals an
- Vergleicht jeden Buchstaben mit dem Alphabet
- Fragt euer Team um Hilfe
- Es ist okay, wenn es schwierig ist – das ist normal!

### "Der Code funktioniert nicht"

**Lösung:**
1. Überprüft, ob ihr den Code richtig eingegeben habt
2. Keine Leerzeichen vor oder nach dem Code
3. Großbuchstaben verwenden (z.B. `SB95` statt `sb95`)
4. Falls es immer noch nicht funktioniert: Fragt den Admin im Chat

### "Die Videos laden nicht"

**Lösung:**
1. Überprüft eure Internetverbindung
2. Wartet ein paar Sekunden
3. Aktualisiert die Seite (F5)
4. Versucht es nochmal

### "Ich bin aus Versehen aus der Seite raus"

**Lösung:**
1. Öffnet die Spelling Bee Seite nochmal
2. Meldet euch mit eurem Team-Code an
3. Eure Fortschritte sind noch da!
4. Ihr könnt weitermachen

### "Ich habe einen Fehler gemacht – kann ich nochmal anfangen?"

**Lösung:**
1. Aktualisiert die Seite (F5)
2. Meldet euch nochmal an
3. Ihr könnt die Wörter nochmal eingeben
4. Es gibt keine Strafe!

### "Der Admin hat den Code geändert"

**Lösung:**
1. Das ist möglich – der Admin kann Codes jederzeit ändern
2. Macht Spelling Bee nochmal
3. Ihr bekommt einen neuen Code
4. Gebt den neuen Code in der Hauptapp ein

## 8️⃣ Tipps & Tricks

### Für Anfänger

- **Langsam starten:** Schaut jedes Video mehrmals an
- **Alphabet nutzen:** Das Gebärden-Alphabet ist eure beste Freundin
- **Zusammen spielen:** Diskutiert mit eurem Team
- **Nicht aufgeben:** Es ist schwierig, aber machbar!

### Für Fortgeschrittene

- **Schneller werden:** Mit Übung erkennt ihr Gebärden schneller
- **Muster sehen:** Manche Buchstaben haben ähnliche Bewegungen
- **Kontext nutzen:** Denkt an religiöse Begriffe
- **Intuition vertrauen:** Manchmal wisst ihr es einfach!

### Zeitmanagement

- **Nicht zu lange:** Spending Bee sollte nicht länger als 15 Minuten dauern
- **Andere Stationen:** Wenn ihr steckenbleibt, macht andere Stationen und kommt später zurück
- **Pausen:** Wenn ihr müde werdet, macht eine kurze Pause
