# 🎭 Bibelpose – Standbild-Rätsel Anleitung

## Was ist Bibelpose?

**Bibelpose** ist eine kreative Station, bei der euer Team eine **biblische Szene als Standbild darstellt**. Ein Admin überprüft das Foto und bestätigt es mit einem Code.

**Punkte:** +50 XP  
**Dauer:** ca. 10-15 Minuten  
**Schwierigkeit:** Mittel  
**Team-Größe:** 3-10 Personen ideal

---

## 📋 Anleitung für Teilnehmer

### Schritt 1: Zur Bibelpose-Station gehen

1. **App öffnen** auf eurem Smartphone/Tablet
2. **Klickt auf die "🎭 Bibelpose" Karte**
3. Eine separate Seite öffnet sich

### Schritt 2: Mit eurem Team-PIN anmelden

1. **Gebt euren 4-stelligen Team-Code ein** (z.B. `A3KX`)
2. **Klickt "🚀 Anmelden"**
3. Die Bibelpose-Seite lädt


### Schritt 3: Biblische Szene auswählen

1. **Ihr seht 20 verschiedene biblische Szenen**
2. **Wählt eine aus, die euch gefällt und scrollt runter**

**Beispiele:**
- Jesus und die Jünger
- David und Goliath
- Die Heilung des Blinden
- Die Auferstehung
- Die Taufe Jesu
- Mose mit den Steintafeln
- Jona und der Wal
- Samson und die Säulen
- Petrus und Jesus auf dem Wasser
- Die Geburt Jesu

### Schritt 4: Standbild darstellen

1. **Euer Team stellt die Szene dar**
2. **Jedes Teamkollege hat eine Rolle**
3. **Versucht, die Szene möglichst realistisch darzustellen**
4. **Nehmt verschiedene Posen an**

**Beispiel: David und Goliath**
```
David:     Kleine Person, Schleuder in der Hand, angespannt
Goliath:   Große Person, Schwert/Lanze, bedrohlich
Zuschauer: Andere Teamkollegen, überrascht/ängstlich
```

### Schritt 5: Foto machen

1. **Macht ein Foto des Standbildes**
2. **Das Foto sollte alle Teamkollegen zeigen**
3. **Gutes Licht ist wichtig!** (nicht zu dunkel)
4. **Klickt auf "📸 Foto hochladen"**

### Schritt 6: Foto hochladen

1. **Wählt das Foto aus eurer Galerie**
2. **Klickt "Foto hochladen"**
3. **Die App lädt das Foto hoch** (kann ein paar Sekunden dauern)

### Schritt 7: Wartet auf Bestätigung

1. **Der Admin überprüft das Foto**
 > Unter dem Reiter Codes > Bibelpose
 > Das Team mit der Anfrage auswählen
 > Runterscrollen und dann auf Bestätigen oder Ablehnen klicken
2. **Das kann ein paar Minuten dauern**
3. **Ihr seht: "⏳ Dein Foto wird gerade überprüft"**

### Schritt 8: Code erhalten oder Ablehnung

**Wenn bestätigt:** ✅
- Ihr seht den Code! 🎉
- Beispiel: `BP42`
- Notiert euch den Code

**Wenn abgelehnt:** ❌
- Ihr seht: "❌ Foto abgelehnt"
- Grund: Die Szene war nicht erkennbar
- Ihr könnt es nochmal versuchen!

### Schritt 9: Code in Hauptapp eingeben

1. **Geht zurück zur Hauptapp**
2. **Klickt auf die "🎭 Bibelpose" Karte**
3. **Gebt den Code ein** (z.B. `BP42`)
4. **Klickt "Bestätigen"**
5. **Station ist erledigt!** ✅ +50 XP
