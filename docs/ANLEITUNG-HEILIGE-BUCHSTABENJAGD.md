# 📜 Heilige Buchstabenjagd – Alphabet-Rätsel Anleitung

## Was ist Heilige Buchstabenjagd?

**Heilige Buchstabenjagd** ist eine Rätsel-Station, bei der euer Team ein **Alphabet mit christlichen Wörtern/Namen ausfüllt**. Ein Admin überprüft das Foto und bestätigt es mit einem Code.

**Punkte:** +50 XP  
**Dauer:** ca. 15-20 Minuten  
**Schwierigkeit:** Mittel  
**Team-Größe:** 2-10 Personen ideal

---

## 📋 Anleitung für Teilnehmer

### Schritt 1: Zur Heilige Buchstabenjagd-Station gehen

1. **App öffnen** auf eurem Smartphone/Tablet
2. **Klickt auf die "📜 Heilige Buchstabenjagd" Karte**
3. Eine separate Seite öffnet sich

### Schritt 2: Mit eurem Team-PIN anmelden

1. **Gebt euren 4-stelligen Team-Code ein** (z.B. `A3KX`)
2. **Klickt "🚀 Anmelden"**
3. Die Heilige Buchstabenjagd-Seite lädt

### Schritt 3: Alphabet ausfüllen

1. **Ihr bekommt ein Alphabet-Blatt** (A-Z)
2. **Für jeden Buchstaben müsst ihr ein christliches Wort oder einen Namen finden**
3. **Ihr könnt bis zu 3 Buchstaben auslassen**

**Beispiele für christliche Wörter:**

| Buchstabe | Beispiele |
|-----------|-----------|
| A | Abraham, Apostel, Altar, Auferstehung, Anbetung |
| B | Bibel, Bethlehem, Barmherzigkeit, Brot, Bund |
| C | Christus, Gemeinde, Gnade |
| D | David, Dankbarkeit, Demut, Diakon |
| E | Engel, Erlösung, Ewigkeit, Evangelium |
| F | Frieden, Freude, Fürsorge, Fürsprache |
| G | Gott, Gnade, Gebet, Gemeinde, Glaube |
| H | Hoffnung, Heiland, Herz, Himmel |
| I | Isa (Prophet), Inspiration |
| J | Jesus, Jünger, Jericho, Jona |
| K | König, Kirche, Kreuz, Kelch |
| L | Liebe, Licht, Lied, Lamm |
| M | Maria, Mose, Mitgefühl, Messias |
| N | Nächstenliebe, Noah, Nächster |
| O | Opfer, Ordnung, Obrigkeit |
| P | Petrus, Psalm, Predigt, Pharisäer |
| Q | Quelle, Quell |
| R | Rettung, Reue, Ruf, Retter |
| S | Segen, Seele, Sünde, Sündenvergebung, Samson |
| T | Tempel, Taufe, Treue, Tisch |
| U | Urteil, Umkehr, Überzeugung |
| V | Vergebung, Vertrauen, Versprechen, Vater |
| W | Wort, Wahrheit, Wunder, Weg |
| X | (schwierig - kann ausgelassen werden) |
| Y | (schwierig - kann ausgelassen werden) |
| Z | Zeichen, Zorn, Zuflucht, Zion |

### Schritt 4: Blatt ausfüllen

1. **Schreibt die Wörter neben die Buchstaben**
2. **Ihr könnt bis zu 3 Buchstaben auslassen** (z.B. X, Y, Q)
3. **Handschrift sollte lesbar sein**
4. **Alle Wörter sollten christlich sein**

**Beispiel:**
```
A - Abraham
B - Bibel
C - Christus
D - David
E - Engel
...
X - (ausgelassen)
Y - (ausgelassen)
Z - Zion
```

### Schritt 5: Foto machen

1. **Macht ein Foto des ausgefüllten Blattes**
2. **Das Foto sollte alle Wörter deutlich zeigen**
3. **Gutes Licht ist wichtig!** (nicht zu dunkel)
4. **Klickt auf "📸 Foto hochladen"**

**Foto-Tipps:**
- ✅ Alle Wörter sollten deutlich lesbar sein
- ✅ Das Blatt sollte vollständig im Foto sein
- ✅ Gutes Licht (Tageslicht oder Lampen)
- ✅ Keine Schatten auf dem Blatt
- ✅ Handschrift sollte lesbar sein
- ✅ Blatt sollte gerade sein (nicht schräg)

### Schritt 6: Foto hochladen

1. **Wählt das Foto aus eurer Galerie**
2. **Klickt "Foto hochladen"**
3. **Die App lädt das Foto hoch** (kann ein paar Sekunden dauern)

### Schritt 7: Wartet auf Bestätigung

1. **Der Admin überprüft das Foto**
2. **Das kann ein paar Minuten dauern**
3. **Ihr seht: "⏳ Dein Foto wird gerade überprüft"**

### Schritt 8: Code erhalten oder Ablehnung

**Wenn bestätigt:** ✅
- Ihr seht den Code! 🎉
- Beispiel: `HB29`
- Notiert euch den Code

**Wenn abgelehnt:** ❌
- Ihr seht: "❌ Foto abgelehnt"
- Grund: Wörter nicht lesbar oder nicht christlich
- Ihr könnt es nochmal versuchen!

### Schritt 9: Code in Hauptapp eingeben

1. **Geht zurück zur Hauptapp**
2. **Klickt auf die "📜 Heilige Buchstabenjagd" Karte**
3. **Gebt den Code ein** (z.B. `HB29`)
4. **Klickt "Bestätigen"**
5. **Station ist erledigt!** ✅ +50 XP
