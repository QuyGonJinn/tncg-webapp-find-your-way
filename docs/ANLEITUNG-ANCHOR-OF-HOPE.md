# ⚓ Anchor of Hope – Hoffnungs-Botschaft Anleitung

## Was ist Anchor of Hope?

**Anchor of Hope** ist eine Schatz-Station, bei der euer Team einen **versteckten Anker findet** und ein **Foto der Hoffnungsbotschaft macht**. Ein Admin überprüft das Foto und bestätigt es mit einem Code.

**Punkte:** +50 XP  
**Dauer:** ca. 10-15 Minuten  
**Schwierigkeit:** Leicht bis Mittel  
**Team-Größe:** 2-10 Personen ideal

---

## 📋 Anleitung für Teilnehmer

### Schritt 1: Armband oder Botschaft fotografieren

1. **Macht ein Foto des Armbands oder der Botschaft**
2. **Das Foto sollte deutlich zeigen, was ihr gefunden habt**
3. **Gutes Licht ist wichtig!** (nicht zu dunkel)

### Schritt 2: Zur Anchor of Hope-Seite gehen

1. QR-Code scannen
2. Eine separate Seite öffnet sich

### Schritt 3: Mit eurem Team-Code anmelden

1. **Gebt euren 4-stelligen Team-Code ein** (z.B. `A3KX`)
2. **Klickt "🚀 Anmelden"**
3. Die Anchor of Hope-Seite lädt

### Schritt 4: Foto hochladen

1. **Klickt auf "📸 Foto hochladen"**
2. **Wählt das Foto aus eurer Galerie**
3. **Klickt "Foto hochladen"**
4. **Die App lädt das Foto hoch** (kann ein paar Sekunden dauern)

### Schritt 5: Wartet auf Bestätigung

1. **Der Admin überprüft das Foto**
 > Unter dem Reiter Codes > Anchor of Hope
 > Das Team mit der Anfrage auswählen
 > Runterscrollen und dann auf Bestätigen oder Ablehnen klicken
2. **Das kann ein paar Minuten dauern**
3. **Ihr seht: "⏳ Dein Foto wird gerade überprüft"**

### Schritt 6: Code erhalten oder Ablehnung

**Wenn bestätigt:** ✅
- Ihr seht den Code! 🎉
- Beispiel: `AH85`
- Notiert euch den Code
- dann auf "zurück klicken, um zum Teilnehmerbereich zurückzukehren"

**Wenn abgelehnt:** ❌
- Ihr seht: "❌ Foto abgelehnt"
- Grund: Der Anker war nicht deutlich sichtbar
- Ihr könnt es nochmal versuchen!

### Schritt 7: Code in Hauptapp eingeben

1. **Klickt auf die "⚓ Anchor of Hope" Karte**
2. **Gebt den Code ein** (z.B. `AH85`)
3. **Klickt "Bestätigen"**
4. **Station ist erledigt!** ✅ +50 XP
