# ⚓ Anchor of Hope – Hoffnungs-Botschaft Anleitung

## Was ist Anchor of Hope?

**Anchor of Hope** ist eine Schatz-Station, bei der euer Team einen **versteckten Anker findet** und ein **Foto der Hoffnungsbotschaft macht**. Ein Admin überprüft das Foto und bestätigt es mit einem Code.

**Punkte:** +50 XP  
**Dauer:** ca. 10-15 Minuten  
**Schwierigkeit:** Leicht bis Mittel  
**Team-Größe:** 2-10 Personen ideal

---

## 📋 Anleitung für Teilnehmer

### Schritt 1: Zur Anchor of Hope-Station gehen

1. **Sucht die Station im Gelände**
2. **Dort findet ihr einen versteckten Anker**
3. **Der Anker hat eine Hoffnungsbotschaft darunter**

**Wo könnte der Anker versteckt sein?**
- ✅ Unter einem Stein
- ✅ An einem Baum
- ✅ In einer Ecke
- ✅ Hinter etwas
- ✅ Auf einem Tisch
- ✅ In einer Box

### Schritt 2: Anker finden und Botschaft lesen

1. **Sucht sorgfältig nach dem Anker**
2. **Lest die Botschaft darunter**
3. **Macht euch Notizen** (falls ihr sie braucht)

**Beispiel-Botschaften:**
- "Gott ist meine Hoffnung"
- "Vertraue auf den Herrn"
- "Hoffnung gibt Kraft"
- "Glaube, Hoffnung, Liebe"
- "Der Anker hält"

### Schritt 3: Armband oder Botschaft fotografieren

1. **Macht ein Foto des Armbands oder der Botschaft**
2. **Das Foto sollte deutlich zeigen, was ihr gefunden habt**
3. **Gutes Licht ist wichtig!** (nicht zu dunkel)

**Foto-Tipps:**
- ✅ Der Anker sollte deutlich sichtbar sein
- ✅ Die Botschaft sollte lesbar sein
- ✅ Gutes Licht (Tageslicht oder Lampen)
- ✅ Keine Schatten auf dem Anker
- ✅ Interessanter Blickwinkel

### Schritt 4: Zur Anchor of Hope-Seite gehen

1. **App öffnen** auf eurem Smartphone/Tablet
2. **Klickt auf die "⚓ Anchor of Hope" Karte**
3. Eine separate Seite öffnet sich

### Schritt 5: Mit eurem Team-PIN anmelden

1. **Gebt euren 4-stelligen Team-Code ein** (z.B. `A3KX`)
2. **Klickt "🚀 Anmelden"**
3. Die Anchor of Hope-Seite lädt

### Schritt 6: Foto hochladen

1. **Klickt auf "📸 Foto hochladen"**
2. **Wählt das Foto aus eurer Galerie**
3. **Klickt "Foto hochladen"**
4. **Die App lädt das Foto hoch** (kann ein paar Sekunden dauern)

### Schritt 7: Wartet auf Bestätigung

1. **Der Admin überprüft das Foto**
2. **Das kann ein paar Minuten dauern**
3. **Ihr seht: "⏳ Dein Foto wird gerade überprüft"**

### Schritt 8: Code erhalten oder Ablehnung

**Wenn bestätigt:** ✅
- Ihr seht den Code! 🎉
- Beispiel: `AH85`
- Notiert euch den Code

**Wenn abgelehnt:** ❌
- Ihr seht: "❌ Foto abgelehnt"
- Grund: Der Anker war nicht deutlich sichtbar
- Ihr könnt es nochmal versuchen!

### Schritt 9: Code in Hauptapp eingeben

1. **Geht zurück zur Hauptapp**
2. **Klickt auf die "⚓ Anchor of Hope" Karte**
3. **Gebt den Code ein** (z.B. `AH85`)
4. **Klickt "Bestätigen"**
5. **Station ist erledigt!** ✅ +50 XP
