# Spelling Bee – Medien-Dateien

## Ordnerstruktur

```
spelling-bee/
├── gebaerden/     → Videos für Gebärden (MP4)
├── bilder/        → Bilder für Buchstaben (JPG oder PNG)
└── memos/         → Sprachmemos (MP3)
```

## Dateinamen

Jede Datei wird nach dem Buchstaben benannt:

| Buchstabe | Gebärde       | Bild          | Sprachmemo    |
|-----------|---------------|---------------|---------------|
| A         | A.mp4         | A.jpg         | A.mp3         |
| B         | B.mp4         | B.jpg         | B.mp3         |
| C         | C.mp4         | C.jpg         | C.mp3         |
| ...       | ...           | ...           | ...           |
| X         | X.mp4         | X.jpg         | X.mp3         |

## Unterstützte Formate

- **Gebärden:** `.mp4`, `.webm`, `.mov`
- **Bilder:** `.jpg`, `.jpeg`, `.png`, `.webp`
- **Sprachmemos:** `.mp3`, `.ogg`, `.wav`

## Hinweis

Wenn eine Datei fehlt, wird ein Platzhalter angezeigt.
